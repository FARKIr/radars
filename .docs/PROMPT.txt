Dáta sú lokálne (statický súbor).

CIEĽ
- Jednostránková aplikácia: “Radary – zoznam miest a lokácií”
- Dáta: zoznam položiek (mesto, cesta, popis miesta, typ merania).
- Funkcie: rýchle vyhľadávanie, filtrovanie, triedenie, skupiny, detail karty, export, štatistiky.
- UI: čisté, čitateľné farby, vysoký kontrast, moderné komponenty, bez preplácania.

TECH STACK
- Next.js (App Router) + React + TypeScript
- Tailwind CSS
- shadcn/ui (Button, Input, Badge, Card, Dialog, Sheet, Tabs, Table, DropdownMenu, Separator, Tooltip, Toast/Sonner)
- @antv/x6 (graf pre „interaktívnu hlavu“)
- Bez databázy, bez Supabase, bez backendu.

DÁTA (vložiť do src/data/radary.ts ako konštantu)
- Každý záznam musí mať: id, mesto, triedaCesty, cesta, lokalita, typMerania
- typMerania enum: “rychlost”, “semafor”, “semafor_rychlost”, “cervena”
- Ak v texte je “semafor + rýchlosť” -> semafor_rychlost
- Ak je “červená na semafore” -> cervena
- Ak je “semafor” -> semafor
- Inak -> rychlost
- „súradnice“ sú v tomto vstupe chýbajúce; do dát pridaj pole suradnice ako null a UI musí jasne ukázať, že nie sú dostupné.

POŽIADAVKY NA UI/UX
1) Horný „hero header“ (interaktívna hlava):
- Panel s @antv/x6 grafom.
- Uzly: kraje/skupiny (Západ, Stred, Východ) + mestá ako uzly.
- Klik na uzol mesta nastaví filter “mesto”.
- Hover zobrazi tooltip s počtom záznamov a typmi merania.
- Musí fungovať na mobile (zmenšený režim: horizontálny scroll, alebo tlačidlo “Zväčšiť” v Sheet/Dialog).

2) Ovládací panel filtrov:
- Vyhľadávanie (fulltext: mesto, cesta, lokalita, typ).
- Filtre: región (Západ/Stred/Východ), triedaCesty (I/xx, II/xxx), typMerania (multi-select).
- Prepínač zobrazenia: “Karty” / “Tabuľka”.
- Triedenie: podľa mesta, podľa triedy cesty, podľa typu.

3) Výpis:
- Karty: výrazné Badge pre typ, mesto veľkým, cesta a lokalita pod tým.
- Tabuľka: stĺpce mesto, triedaCesty, lokalita, typ.
- Klik na položku otvorí detail (Dialog/Sheet): všetky polia + “Kopírovať text” (to clipboard) + “Kopírovať JSON”.

4) Súhrny a štatistiky:
- Počet záznamov po filtroch.
- Počty podľa typMerania (mini karty).
- Počty podľa regiónu (mini karty).

5) Export:
- Tlačidlo “Export CSV” (client-side generovanie).
- Tlačidlo “Export JSON”.

6) Prístupnosť a kvalita:
- Klávesové ovládanie, focus ring, kontrast.
- Bez console.log v produkcii.
- Bez magických čísel v logike (konštanty).
- Priebežne typovo striktne (žiadne any/as/unknown).
- Čistá štruktúra komponentov, bez duplicít.
